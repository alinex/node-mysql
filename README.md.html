¶ Package: alinex-mysql

A simple mysql access wrapper which will leverage the integration of a mysql database into an alinex based application.

  • full configurable
  • connection pooling
  • integrated debug possibilities
  • short method calls

It is one of the modules of the Alinex Universe following the code standards defined there.

¶ Install

The easiest way is to let npm add the module directly:

> npm install alinex-mysql --save
